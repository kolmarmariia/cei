Данные выгружаются по каждому дню по каждому серверу.
            Название файлов имеет формат:
                2020 – год;
                08 – месяц; 
                01 – число;
                s2 – сервер №2
          В каждом архиве содержатся 4 файла в соответствии с таблицами в ТЗ.


1.      Файл 2020-08-01_s2\2020-08-01_s2.csv – таблица 1
        Поля:
            pk_id                    - код перевозчика
            pk_name             - название перевозчика


2.      Файл 2020-08-01_s2\2020-08-01_s2_2.csv – таблица 2
          Поля:
            tr_id                     - уникальный идентификатор транспортного средства
            pk_id                   - код перевозчика
            tt_id                     - код типа транспортного средства
            tr_garagnum      - гаражный (инвентарный) номер транспортного средства
            tr_statenum       - государственный номер транспортного средства
 	    tr_class		- класс ТС 

3.      Файл 2020-08-01_s2\2020-08-01_s2_3.csv – таблица 3
          Поля:
            tr_id                     - уникальный идентификатор транспортного средства
            mr_id                   - код маршрута
            mr_name            - номер маршрута
            graph                   - номер графика
            tabelnum            - табельный номер водителя (может быть не заполнено)
            condnum            - табельный номер кондуктора (в системе не ведется всегда 0)
            date                     - дата наряда
            startshift             - время начала смены
            endshift               - время окончания смены

4.      Файл 2020-08-01_s2\2020-08-01_s2_4.csv – таблица 4
           Поля:
            tr_id                     - уникальный идентификатор транспортного средства           
            mr_id                   - код маршрута
            mr_name            - номер маршрута
            graph                   - номер графика
            tabelnum            - табельный номер водителя (может быть не заполнено)
            condnum            - табельный номер кондуктора (в системе не ведется всегда 0)
            timenav               - навигационное время (локальное Самарское)
            long                      - географическая долгота
            lat                         - географическая широта
            speed                   - скорость (км/ч)
            mark                    - особые отметки (всегда 0)
